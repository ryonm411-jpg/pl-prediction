# Template usage

- Copy `templates/TEMPLATE_MATCH.md` into a match file under match_analysis/<season>/<competition>/
  e.g. match_analysis/2025-26/EPL/2026-02-01_EPL_ARS_AVL.md

- Fill the fields manually. Do NOT change the template headings.

- Use `templates/TEMPLATE_TEAM.md` to create or update team profile files under `teams/`.

This folder contains templates only. Do not add match data here.
